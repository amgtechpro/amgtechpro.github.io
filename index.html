HTML Tags - Fundamentals

Tag Name Description Example
<html> Root Element This is the root element of an HTML document. All other HTML elements are contained within this tag. <html>
(The entire HTML content goes here)
</html>
<head> Head Contains meta-information about the HTML document, such as its title, character set, and links to external files (like CSS). <head>
<title>Page Title</title>
</head>
<title> Title Defines the title of the document, which appears in the browser's title bar or tab. <title>My Webpage</title>
<body> Body Contains the content of the webpage, such as text, images, links, and other elements. <body>
(Visible content goes here)
</body>
<h1> to <h6> Headings Defines headings, with <h1> being the most important and <h6> the least. Headings structure the content. <h1>Main Heading</h1>
<h2>Subheading</h2>
<p> Paragraph Defines a paragraph of text. <p>This is a paragraph of text.</p>
<a> Anchor (Link) Creates a hyperlink that can link to another webpage or resource. The href attribute specifies the destination. <a href="https://example.com">Click Here</a>
<ul>, <ol> and <li> Lists <ul> defines an unordered list (bulleted), and <ol> defines an ordered list (numbered). <li> defines each list item. <ul>
<li>Item 1</li>
<li>Item 2</li>
</ul>
<img> Image Embeds an image into the webpage. The src attribute defines the path to the image, and alt provides alternative text. <img src="image.jpg" alt="Description of Image">
